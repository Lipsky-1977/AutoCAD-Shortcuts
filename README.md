# AutoCAD 2027 快捷键查询

一个纯 HTML 快捷键查询工具，**无需安装任何依赖**，手机和 PC 浏览器均可直接打开使用。

## 使用方式

直接打开 `index.html` 即可，或将文件传到手机浏览器中。

## 功能

- **159 条命令别名与快捷键**，覆盖 11 个常用分类
- **实时搜索**：按命令别名（如 L、PL、TR）或功能名称搜索
- **分类筛选**：绘图工具、修改工具、功能键 (F1-F12)、Ctrl 组合键等
- **深色模式**：跟随系统主题自动切换
- **触控优化**：移动端单手操作适配
- **离线可用**：单文件，不依赖网络

## 快捷键分类

| 分类 | 说明 |
|------|------|
| 绘图工具 | L(直线)、PL(多段线)、C(圆)、REC(矩形) 等 |
| 修改工具 | CO(复制)、MI(镜像)、TR(修剪)、F(圆角) 等 |
| 图层与属性 | LA(图层)、MO(属性)、MA(匹配) 等 |
| 标注与文字 | DLI(线性)、DAL(对齐)、MT(文字) 等 |
| 视图控制 | Z(缩放)、P(平移)、3DO(三维观察) 等 |
| 功能键 | F1(帮助)~F12(动态输入) |
| Ctrl 组合键 | Ctrl+S(保存)、Ctrl+P(打印) 等 |
| 工具与设置 | OP(选项)、UN(单位)、PURGE(清理) 等 |
| 选择与查询 | DI(距离)、AREA(面积)、LI(列表) 等 |
| 打印与输出 | PLOT、PDF、JPGOUT 等 |
| 三维工具 | EXT(拉伸)、REV(旋转)、UNI(并集) 等 |
